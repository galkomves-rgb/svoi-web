# AI AGENTS SYSTEM (SVOI PLATFORM)

## 🎯 Goal
Build a scalable, clean, user-friendly platform without losing existing functionality.
Improve structure, UX, and UI system.

---

# 🧠 AGENT ROLES

## 1. Codex (Primary Executor)
Role:
- Implements changes in code
- Refactors structure
- Works across multiple files
- Applies UI system

Responsibilities:
- DO NOT remove existing modules, routes, or buttons
- Make targeted improvements, not blind rewrites
- Follow UI/UX rules strictly
- Keep project stable

---

## 2. Copilot (Inline Assistant)
Role:
- Helps with small edits
- Autocomplete
- Local improvements

Responsibilities:
- Improve readability
- Fix small issues
- Do NOT change architecture

---

## 3. Grok (Research & Critical Thinking)
Role:
- Analyze UX/UI
- Suggest improvements
- Find alternative solutions
- Critically evaluate decisions

Responsibilities:
- DO NOT write final code
- Focus on reasoning and structure
- Provide options, not commands

---

## ⚙️ AUTO MODE RULES (VERY IMPORTANT)

When multiple agents are used:

1. Grok → THINK
2. Codex → BUILD
3. Copilot → REFINE

NEVER:
- Let multiple agents rewrite the same thing blindly
- Remove functionality for simplicity
- Change logic without explanation

---

# 🧱 PRODUCT RULES

- Preserve ALL important functionality
- City = filter, not core concept
- Platform ≠ 2 cities only
- UI language = Ukrainian
- Tech language = English (code)

---

# 🎨 UI/UX RULES

## Structure
- One screen = one main purpose
- Reduce vertical scroll
- Avoid stacking cards everywhere

## Components
- Use dropdowns instead of open inputs
- Use filters instead of long forms
- Use quick actions

## Visual
- Avoid sterile UI
- Add hierarchy, spacing, icons
- Keep interface warm but clean

---

# 🚫 WHAT NOT TO DO

- Do not delete modules
- Do not oversimplify by removing features
- Do not use cards for everything
- Do not create giant spacing
- Do not break existing flows

---

# ✅ SUCCESS CRITERIA

- Platform feels like product, not MVP
- Less scrolling
- Better hierarchy
- Structured forms
- Clean UI system
- All functionality preserved
