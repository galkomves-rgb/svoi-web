# svoi-web

SVOI — platform for Ukrainian communities. Local services, listings, events and support.

## Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Playwright for smoke validation
- Mock data only, no backend

## Routes

- `/`
- `/start`
- `/[city]`
- `/[city]/listings`
- `/[city]/events`
- `/[city]/services`
- `/[city]/guide`

Initial city slugs:

- `torrevieja`
- `alicante`

## Install

```sh
npm install
```

If Playwright browsers are not installed yet:

```sh
npx playwright install
```

## Dev

```sh
npm run dev
```

Local app:

```text
http://127.0.0.1:3000
```

## Build

```sh
npm run build
```

## Production Preview

```sh
npm run serve
```

## Smoke Tests

```sh
npm run smoke
```

## Deploy

Recommended target: Vercel.

1. Push the repository to GitHub.
2. Import the project into Vercel.
3. Keep the default Next.js framework preset.
4. Use the default build command:

```text
npm run build
```

5. Use the default output generated by Next.js.

## Structure

- `app` - App Router pages and layouts
- `components` - reusable UI and layout components
- `data` - local mock data
- `lib` - route and data helpers
- `tests` - Playwright smoke tests
- `.vscode` - local tasks and launch configuration
- `docs` - product and technical documentation

## Notes

- This project stays frontend-only.
- No backend, auth, or real submission logic is implemented.
- The setup is ready for Vercel deployment and future backend integration.
